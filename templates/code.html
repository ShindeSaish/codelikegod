Advanced Python

  • List Comprehension: [x*2 for x in range(10) if x%2 == 0]
  • Lambda Functions: lambda x: x + 10
  • Generator Expression: (x**2 for x in range(5))
  • Decorators: def decorator(func): return wrapper(func)
  • Context Manager: with open('file.txt', 'r') as f:
  • List Slicing: my_list[::2]
  • Unpacking: a, *b, c = range(5)

Advanced Java Cheat Sheet

  • Streams: Stream.of(1, 2, 3).map(x -> x * 2).forEach(System.out::println);
  • Lambda Expressions: Comparator cmp = (a, b) -> a - b;
  • Method References: list.forEach(System.out::println);
  • Generics: List list = new ArrayList<>();
  • Optional Class: Optional opt = Optional.ofNullable(value);
  • Functional Interface: public interface MyInterface { void doSomething(); }
  • Stream Collectors: Collectors.toList()

Advanced JavaScript Cheat Sheet

  • Arrow Functions: const add = (a, b) => a + b;
  • Destructuring Assignment: const {name, age} = person;
  • Promises: fetch('url').then(response => response.json()).then(data => console.log(data));
  • Async/Await: async function fetchData() { let response = await fetch(url); }
  • Spread Operator: const newArray = [...arr1, ...arr2];
  • Rest Parameters: function sum(...args) { return args.reduce((a, b) => a + b, 0); }
  • Event Delegation: parentElement.addEventListener('click', function(e) { if (e.target.matches('.child')) { ... } });
